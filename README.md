# Python for data analysis
This repository is a structured Python 3 learning process, built from the book Python for Data Analysis by Wes McKinney with a focus on:
- Clean code
- Data science foundations
- Reproducible learning
This repository is projects oriented, with the aim to apply the concepts and techniques learned in the book to real-world data analysis tasks. The book mentioned above provides asts as a reference. Each project will be accompanied by a Jupyter Notebook that provides an interactive learning experience, as well as a Python script that contains clean and well-documented code.
## Structure
- `python-for-data-analysis/`: Interactive lessons and explanations and my reflection on the learning process
- The rest of the files are projects: Each project will have a Jupyter Notebook and a Python script version.

## Projects
- `project1_news_api_counting_statistics/`: This project uses the News API to obtain data about news articles mentioning the any keywords and saves it to a CSV file. It then counts the occurrences of words in the article titles and visualizes the results using a bar chart.
